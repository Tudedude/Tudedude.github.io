Biography

I am a 20 year-old computer science student, living in the dense forests of northern California. I have been programming since I was 10 years old, and have a diverse range of experience and interests. I began my journey when my father, a web developer, began teaching me basic HTML and CSS. Quickly I became enamored with the experience of creating my own digital projects, and avidly learned fundamental XHTML, CSS, and JavaScript. Then, my enjoyment of games led me to Java, where I learned to create modifications, server plugins, and more for the popular game Minecraft. Since then, I have worked in a variety of languages including:
  • HTML
  • CSS
  • JavaScript
  • Java
  • PHP
  • JSON
  • XML
  • YAML
I have also utilized technologies including:

  • React
  • Android SDK
